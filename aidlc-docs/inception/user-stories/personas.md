# User Personas (축소판)

## Persona 1: Customer (고객)
- **역할**: customer
- **목표**: 상품 탐색, 주문, 리뷰 작성
- **Pain Points**: 리뷰 참고 불가, 모바일 UI 불편

## Persona 2: Admin-A (Super Admin)
- **역할**: admin_a
- **목표**: 전체 관리 (회계 포함), 인플루언서 커미션 정산, Admin 계정 관리
- **접근**: 전체 (회계/재무 + 운영)

## Persona 3: Admin-B (Operations Admin)
- **역할**: admin_b
- **목표**: 상품/주문/재고 관리, 인플루언서 협업 운영
- **접근**: 운영만 (회계 제외)

## Persona 4: Influencer (인플루언서)
- **역할**: influencer
- **목표**: 프로필 등록, 협업 확인, 레퍼럴 링크 생성, 커미션 조회
